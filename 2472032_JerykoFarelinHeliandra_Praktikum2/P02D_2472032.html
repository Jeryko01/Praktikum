Pengenlan HTML, CSS, JavaScript, PHP, Node.js. Ruby

HMTL (Hyper Text Markup Language) adalah bahasa markup untuk membuat struktur halaman web. HTML menggunakan tag-tag khusus untuk mendefinisiakan elemen-elemen web.


CSS (Cascading Style Sheet) digunkaan untuk mendesain dan mempercantik halaman web. Dengan CSS, anda dapat mengatur wrna, font, layout, dan banyak lagi.



JavaScript memebrikan interaktivitas pada halaman web. Misalnya, anda dapat membuat animasi, validasi formulir, atau fitur dinamis lainnya menggunkan JavaScript.



PHP (Hypertext Preproccessors) adalah bahasa pemrogramanb server-side yang digunakan untuk mengembangkan aplikasi web dinamis. PHP sering digunakan seperti MySQL untuk menyimpamn dan mengambil data.




Node.js adalah platform runtime JavaScript yang berjalan di server. Node.js memungkinkan pengembang untuk membuat aplikasi backend menggunakan JavaScript, sehingga memudahkan integrasi antara frontend dan backend.




Ruby adalah bahasa pemrograman yang sederhana dan fleksibel, sering digunakan bersama framework Rail (Ruby on Rails) untuk pengembangan wdeb. Ruby dikenal karena sintaksnya yang mudah dipahami danproduktivitasnya yang tinggi.
